# AnimeWorld – Project Description

**AnimeWorld** is a responsive anime streaming website developed using **HTML5, CSS3, Bootstrap 5, and JavaScript**. The project is designed to provide an attractive and user-friendly interface where anime fans can explore trending, popular, and latest anime content.

The website features a modern homepage with a **video banner**, responsive **navigation bar**, **search functionality**, anime categories, trending and popular sections, premium subscription plans, and a contact page. It also includes visually appealing anime cards with hover effects, smooth animations, and a fully responsive layout that works across desktop, tablet, and mobile devices.

The main objective of this project is to demonstrate front-end web development skills, responsive design principles, and clean user interface development. AnimeWorld showcases the effective use of Bootstrap components, custom CSS styling, and JavaScript to create an engaging and interactive user experience.

**Technologies Used:**

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

**Key Features:**

* Responsive navigation bar
* Hero section with video background
* Search bar
* Trending, Popular, and Movies sections
* Interactive anime cards with hover effects
* Premium subscription plans
* Contact page
* Mobile-friendly responsive design
* Clean and modern UI
